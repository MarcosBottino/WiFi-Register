Este proyecto fue creado con el fin de ser utilizado en un Hotspot WiFi. Funciona de la siguiente manera: 

Los usuarios cuando quieren ingresar al WiFi, ingresan sus datos para luego obtener acceso y esos datos se almacenan en un Hoja de Calculo 
para futuros usos de publicidad. 

Para que eso funcione, utilice una API que se denomina SheetDB (Acceso: https://sheetdb.io/), la cual se utiliza creando una Hoja de Calculo desde la web. Una vez extraida en SheetDB 
se convertira en un link el cual ingresaremos en el "action" del <Form>.

Espero que les sirva y guste.

This proyect was made with the purpose of being use in a WiFi Hotspot. This is how it works:

The user when they want to use Wifi, enter their data for access and those data will be stored in a SpreadSheet. 

This SpreadSheet works with an API, SheetDB (Access: https://sheetdb.io/), which is use creating a SpreadSheet in the web. Once extract will become a link that we will use inside the <form>
in a "action".
